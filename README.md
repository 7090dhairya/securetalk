Secure Talk is a real-time private messaging backend built using Node.js, Express, MongoDB, and WebSockets with session-based authentication.

features:---
User registration and login (bcrypt password hashing)
Session-based authentication
Authenticated WebSocket connections
Private real-time messaging
MongoDB message persistence
Online user tracking
REST + WebSocket hybrid architecture


tech stak i used:---
Node.js
Express.js
MongoDB
WebSocket (ws)
bcrypt
express-session
cookie-parser
